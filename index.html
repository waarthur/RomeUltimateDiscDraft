🗄️ Supabase Database Setup
✅ Connected to Supabase — registrations and games are stored in the cloud.
⚠ Database Not Yet Connected

Follow these steps (≈5 min) to enable cloud storage so players on any device can register and game scores are saved.

1
Create a free Supabase account
Go to supabase.com → Start your project. Sign up free.
2
Create a new project
Click New Project. Name it discdraft. Choose nearest region. Click Create new project.
3
Run this SQL (creates both tables)
Click SQL Editor in the sidebar, paste and run: 
-- Players table
create table players (
  id bigint generated always as identity primary key,
  first text not null, last text not null,
  age integer not null, email text, talent integer not null,
  event_id text not null, registered_at timestamptz default now()
);
alter table players enable row level security;
create policy "Public read" on players for select using (true);
create policy "Public insert" on players for insert with check (true);
create policy "Public delete" on players for delete using (true);

-- Games table
create table games (
  id bigint generated always as identity primary key,
  event_id text not null,
  played_at timestamptz default now(),
  teams jsonb not null,
  scores jsonb not null
);
alter table games enable row level security;
create policy "Public read" on games for select using (true);
create policy "Public insert" on games for insert with check (true);
create policy "Public delete" on games for delete using (true);
4
Copy your API credentials
Go to Project Settings → API. Copy your Project URL and anon / public key. Paste below, then click Test Connection.

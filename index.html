🔗 Deploy & Share
💡 This app is a single HTML file. To share a registration link with your group, you need to host it online for free using GitHub Pages. Follow the steps below — no coding experience needed.
1
Create a free GitHub account
Go to github.com and sign up for a free account if you don't have one. Confirm your email address.
2
Create a new repository
Click the + icon (top-right) → New repository.
Name it something like discdraft.
Set it to Public. Check "Add a README file". Click Create repository.
3
Upload the app file
Inside your new repository, click Add file → Upload files.
Drag and drop the discdraft.html file you downloaded.
Important: rename it to index.html before uploading (or rename it after by clicking the file → pencil icon).
Click Commit changes.
4
Enable GitHub Pages
Go to your repository → Settings → Pages (left sidebar).
Under Source, select Deploy from a branch.
Set branch to main and folder to / (root). Click Save.
Wait ~60 seconds, then refresh — your live URL will appear at the top of that page.
5
Your live URL will look like this:
https://YOUR-USERNAME.github.io/discdraft/

Paste it below to generate a ready-to-share group chat message. ↓
